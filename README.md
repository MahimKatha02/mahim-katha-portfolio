## рџљЂ Installation

Clone the repository:

```bash
git clone https://github.com/MahimKatha02/mahim-katha-portfolio.git
```

Move into the project directory:

```bash
cd mahim-katha-portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

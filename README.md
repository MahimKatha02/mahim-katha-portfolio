This portfolio website represents my professional journey in Machine Learning, Artificial Intelligence, Data Science, Computer Vision, Software Development, and Networking.

The goal of this project is to provide a centralized platform where recruiters, researchers, collaborators, and clients can explore my technical skills, featured projects, certifications, and professional services.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/MahimKatha02/mahim-katha-portfolio.git
```

Move into the project directory:

```bash
cd mahim-katha-portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Create production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

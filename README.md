# Installation

Clone the repository

```bash
git clone https://github.com/MahimKatha02/mahim-katha-portfolio.git
```

Move into the project

```bash
cd mahim-katha-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
